# Credit Card Dispute Resolution System — Digital Transformation
### Business Analyst Portfolio Project | BFSI Domain | Canadian Banking

---

## Project Overview
This project presents a complete Business Analysis for the digital 
transformation of a credit card dispute resolution system at a 
mid-size Canadian bank (MapleTrust Bank — fictional).

The current manual process takes 14 business days to resolve disputes,
violates FCAC (Financial Consumer Agency of Canada) guidelines,
and costs an estimated $4.5M annually in operational inefficiencies.

This BA project defines the full solution — from requirements through 
to test cases — targeting a 5-day resolution cycle and $2.7M in 
annual savings.

---

## Business Problem
- 45,000 disputes processed annually — 100% via phone, no self-service
- Average resolution: 14 days vs FCAC guideline of 5 days
- Customer satisfaction: 32% vs industry benchmark of 72%
- No API integration with Visa/Mastercard — manual fax/email process
- Zero audit trail — regulatory compliance risk

---

## Proposed Solution
A digital dispute resolution platform featuring:
- 24/7 customer self-service portal (web + mobile)
- Automated rule-based dispute categorisation engine
- Single-screen agent workbench
- Real-time Visa/Mastercard API integration
- Automated SMS/email notification system
- Management analytics dashboard
